Switch Statements

switch(value) {
case 1:
// code
break;
case 2:
// code
break;
default:
// code
}

Switch Statement Example

let day = 3;
switch(day) {
case 1:
console.log("Monday");
break;
case 2:
console.log("Tuesday");
break;
case 3:
console.log("Wednesday");
break;
default:
console.log("Another day");
}

Switch statement breakdown:

  • switch(value) - Expression to evaluate
  • case - Each possible value to match
  • break - Exits the switch block when matched
  • default - Runs if no cases match

This example will output "Wednesday" because day equals 3. Without break statements, it would continue executing subsequent cases.

Loops

For Loop

for (let i = 0; i < 10; i++) {
        // logic here - this is where your code goes to be repeated
    }

In the parentheses:
- let i = 0: Initialize counter variable
- i < 10: Condition to keep loop running
- i++: Increment counter after each iteration. i++ = i = i + 1. i-- = i = i - 1

Example Loop

let sum = 0;
for (let i = 1; i <= 5; i++) {
    sum += i;    // Adds numbers 1 through 5
}
// Result: sum equals 15

This loop calculates the sum of numbers from 1 to 5 (1 + 2 + 3 + 4 + 5 = 15)

Functions

Basic Function

// Function declaration
function addNum(num1, num2) {
    return num1 + num2;
}

// Function usage
const result = addNum(5, 3);    // returns 8
const sum = addNum(10, 20);     // returns 30

Function breakdown:

  • function - Keyword to declare a function
  • addNum - Function name using camelCase
  • (num1, num2) - Parameters the function accepts
  • return - Specifies what value to send back

This function takes two numbers as input and returns their sum. It can be reused multiple times with different values.

Constructor Functions

// Constructor function
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.greet = function() {
        return `Hi, I'm ${this.name}`;
    }
}

// Creating new instances
const person1 = new Person("John", 30);
const person2 = new Person("Jane", 25);

console.log(person1.name);     // "John"
console.log(person2.age);      // 25
console.log(person1.greet());  // "Hi, I'm John"

Constructor functions are used as templates to create objects:

  • function Person() - Constructor function (capitalized by convention)
  • this - Refers to the new instance being created
  • new Person() - Creates a new object from the function 'Person'
  • .greet() - Method that can be called on instances

DOM Manipulation

Editing Elements

  • .remove() - Removes the element from DOM
  • .innerText - Gets or sets the visible text content (without HTML tags)
  • .textContent - Gets or sets text content
  • .style - Changes CSS styles directly on element

Examples

// Remove example
        const element = document.querySelector('.some-class');
        element.remove();  // Removes element from the page

        // textContent example
        const heading = document.querySelector('h1');
        heading.textContent = 'New Heading';  // Changes text content
        console.log(heading.textContent);     // Gets text content

        // innerText example
        const div = document.querySelector('div');
        div.innerText = 'Hello World';  // Changes visible text
        console.log(div.innerText);     // Gets visible text only

        // style example
        const box = document.querySelector('.box');
        box.style.backgroundColor = 'red';  // Changes background color
        box.style.padding = '20px';        // Adds padding
        box.style.border = '2px solid black';

    • Event Listeners

    Basic Syntax

    element.addEventListener(eventType, callback)

    Common Event Types

    • 'click' - Mouse click
    • 'submit' - Form submission
    • 'keydown' - Key press
    • 'mouseover' - Mouse hover
    • 'load' - Page/element loaded

    Complete Example

    // Button click example
const button = document.querySelector('#submitBtn');
button.addEventListener('click', function(event) {
    event.preventDefault();  // Prevents default behavior
    console.log('Button clicked!');
});

// Form submission example
const form = document.querySelector('#myForm');
form.addEventListener('submit', function(e) {
    e.preventDefault();  // Stops form from submitting
    
    // Get input value
    const input = document.querySelector('#username');
    console.log('Form submitted with:', input.value);
});

// Keyboard event example
document.addEventListener('keydown', function(event) {
    console.log('Key pressed:', event.key);
    
    // Check for specific key
    if (event.key === 'Escape') {
        console.log('Escape key pressed!');
    }
});

// Mouse hover example
const box = document.querySelector('.box');
box.addEventListener('mouseover', function() {
    this.style.backgroundColor = 'red';
});
box.addEventListener('mouseout', function() {
    this.style.backgroundColor = 'blue';
});

    Event Object Properties

    • event.target - Element that triggered the event
    • event.preventDefault() - Stops default behavior
    • event.key - Key pressed (for keyboard events)
    • event.clientX/clientY - Mouse coordinates

    Using Arrow Functions

    // Modern syntax with arrow function
element.addEventListener('click', (e) => {
    console.log('Clicked:', e.target);
});

    Note: The this keyword behaves differently in arrow functions. Use event.target instead of this when using arrow functions.
